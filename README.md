# Expense Tracker Bot

Telegram-бот для личного учёта расходов. Всё хранится локально в SQLite.  
Никаких внешних API — только Telegram.

---

## Быстрый старт

### 1. Создай бота в Telegram

1. Открой [@BotFather](https://t.me/BotFather) в Telegram.
2. Отправь `/newbot`.
3. Введи имя бота (например, `My Expense Bot`).
4. Введи username (например, `my_expense_bot` — должен заканчиваться на `bot`).
5. BotFather пришлёт токен вида `7123456789:AAF…` — скопируй его.

### 2. Настрой окружение

```bash
cd expense-bot
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Заполни `.env`

```
TELEGRAM_BOT_TOKEN=7123456789:AAF…
```

### 4. Запусти бота

```bash
python bot.py
```

Бот начнёт polling. Оставь терминал открытым.  
Для фонового запуска: `nohup python bot.py &`

## Как пользоваться

### Добавление расхода

Просто пиши описание и сумму — порядок неважен:

```
массаж 5000
продукты 3200 рублей
бензин 4500 руб
```

Бот распарсит сумму и попросит выбрать категорию и способ оплаты через кнопки.

**Пример диалога:**

```
Ты:   бензин 4500 руб

Бот:  💰 бензин — 4 500 руб.

      Выбери категорию:
      [🏠 Домашние]  [💼 Рабочие]  [👶 Дети]

Ты:   [нажал 🏠 Домашние]

Бот:  Категория: 🏠 Домашние

      Выбери способ оплаты:
      [💵 Наличные]  [💳 ДВ]  [💳 KG Rus]

Ты:   [нажал 💳 ДВ]

Бот:  ✅ Записал: бензин — 4 500 руб. | Домашние | ДВ | 17.05.2026
```

---

### Просмотр расходов

| Что напиши | Что вернёт бот |
|---|---|
| `расходы за март` | Все расходы за март |
| `все за апрель 2026` | Все расходы за апрель 2026 |
| `домашние за март` | Расходы категории «Домашние» |
| `рабочие за февраль` | Расходы категории «Рабочие» |
| `дети за январь` | Расходы категории «Дети» |
| `наличные за март` | Оплачено наличными |
| `дв за апрель` | Оплачено картой ДВ |
| `kg rus за март` | Оплачено картой KG Rus |
| `итого за март` | Сводка по категориям и оплате |
| `последние 10` | Последние 10 записей |

**Пример отчёта по категории:**

```
📊 🏠 Домашние | Март 2026

1. 05.03 — Массаж — 5 000 руб. [Наличные]
2. 12.03 — Продукты — 3 200 руб. [ДВ]
3. 28.03 — Химчистка — 1 800 руб. [KG Rus]
────────────── 💰 Итого: 10 000 руб.
```

**Пример сводки (`итого за март`):**

```
📊 Итого за Март 2026

🏠 Домашние: 10 000 руб.
💼 Рабочие: 25 000 руб.
👶 Дети: 7 500 руб.

💵 Наличные: 15 000 руб.
💳 ДВ: 20 000 руб.
💳 KG Rus: 7 500 руб.
━━━━━━━━━━━━━━
💰 Всего: 42 500 руб.
```

### Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и краткая инструкция |
| `/help` | Примеры запросов |
| `/last` | Последние 10 расходов |
| `/stats` | Итого за текущий месяц |
| `/cancel` | Отменить текущий ввод расхода |

## Структура проекта

```
expense-bot/
├── bot.py          # точка входа, сборка Application
├── handlers.py     # все Telegram-хэндлеры
├── database.py     # асинхронные операции с SQLite
├── parser.py       # парсинг суммы и текстовых запросов (regex)
├── reports.py      # форматирование отчётов
├── config.py       # загрузка .env
├── requirements.txt
└── README.md
```

База данных `expenses.db` создаётся автоматически при первом запуске.

## Известные ограничения

- Если описание расхода содержит слово месяца и слово `расходы`/`все`/`список`, бот может перепутать его с запросом отчёта. Пример: `"купил все вещи за март 1500"` — в этом случае лучше написать `вещи март 1500`.
- Год по умолчанию — текущий. Для другого года добавь его явно: `итого за март 2025`.
